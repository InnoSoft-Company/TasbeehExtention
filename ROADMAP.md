# 🚀 الخطة التطويرية الشاملة - الإصدار القادم (v2.0.0)

هذا الملف يوضح الرؤية التفصيلية لنقل إضافة **Tasbeeh Reminder** من مجرد أداة تنبيه بسيطة إلى منصة متكاملة للذكر داخل VS Code.

---

## 1. تطوير واجهة المستخدم (UI/UX)
### 🔹 أيقونة شريط الحالة (Status Bar Item)
- **الوصف**: إضافة أيقونة دائمة في شريط الحالة بالأسفل.
- **التفاصيل**:
    - يظهر بجانبها عداد تنازلي (مثلاً: `📿 12m`) يوضح الوقت المتبقي للتنبيه القادم.
    - عند الضغط عليها، تظهر قائمة سريعة (Quick Pick) تسمح بـ:
        - عرض ذكر الآن.
        - إيقاف مؤقت للتنبيهات (Snooze).
        - تغيير المدة الزمنية بسرعة.

### 🔹 لوحة التحكم الرسومية (Webview Dashboard)
- **الوصف**: صفحة إعدادات متطورة بديلة عن الإعدادات التقليدية لـ VS Code.
- **التفاصيل**: تصميم واجهة بـ HTML/CSS تتيح للمستخدم اختيار "ثيمات" الأذكار، ورؤية إحصائيات يومية لعدد الأذكار التي تمت قراءتها.

---

## 2. نظام التنبيهات المطور
### 🔹 نظام الفئات (Categories)
- **الوصف**: تقسيم الأذكار إلى حزم.
- **الحزم المقترحة**:
    - **أذكار الاستغفار**: تركز على طلب المغفرة.
    - **أذكار التسبيح**: سبحان الله، الحمد لله، الخ.
    - **أذكار الصباح والمساء**: تظهر تلقائياً في أوقاتها المناسبة (حسب توقيت الجهاز).
    - **آيات قرآنية**: تنبيهات بآيات قصيرة ومؤثرة.

### 🔹 التنبيهات الصوتية (Audio Feedback)
- **الوصف**: إضافة أصوات خفيفة اختيارية.
- **التفاصيل**:
    - خيارات لأصوات طبيعية (زقزقة عصافير، صوت ماء هادئ) أو أصوات تنبيه "Minimalist".
    - إمكانية كتم الصوت بضغطة واحدة من شريط الحالة.

---

## 3. الذكاء والإنتاجية
### 🔹 العداد الذكي (Smart Counter)
- **الوصف**: تسجيل عدد المرات التي ضغط فيها المستخدم على "تم الذكر".
- **التفاصيل**: 
    - تخزين الإحصائيات محلياً (Global State).
    - عرض "مستوى التقدم" (مثلاً: أنت ذاكر اليوم بنسبة 80% من هدفك اليومي).

### 🔹 وضع التركيز (Focus Mode Integration)
- **الوصف**: التوقف عن التنبيه تلقائياً إذا كان المستخدم في وضع "Do Not Disturb" في VS Code أو أثناء عرض تقديم (Presentation).

---

## 4. الجانب التقني (Refactoring)
- **الانتقال إلى البرمجة الكائنية (OOP)**: إعادة كتابة `extension.js` باستخدام `Classes` لتسهيل إضافة الميزات الجديدة بدون تعقيد الكود.
- **نظام القوالب (Templates)**: السماح للمستخدمين بتصميم شكل رسالة التنبيه (إذا سمح VS Code بذلك مستقبلاً) أو استخدام ألوان مخصصة.
