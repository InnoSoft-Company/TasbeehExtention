# 📿 Tasbeeh Reminder | ذكر الله

**ألا بذكر الله تطمئن القلوب** - إضافة ذكية لبرنامج VS Code تهدف إلى مرافقة المبرمج المسلم في رحلته اليومية، لتذكره بذكر الله وتجعله رطباً بلسانه حتى في أوج تركيزه البرمجي.

---

## ✨ المميزات الحالية (v1.0.1)
- **🔔 تنبيهات دورية**: ظهور أذكار منبثقة بشكل هادئ لا يقطع تركيزك.
- **⚙️ تحكم كامل**: إمكانية تخصيص المدة الزمنية (Interval) بين كل ذكر وآخر.
- **📝 قائمة مخصصة**: يمكنك إضافة أذكارك الخاصة أو تعديل القائمة الافتراضية من الإعدادات.
- **🚀 تفعيل فوري**: رسالة ترحيبية تؤكد عمل الإضافة فور تشغيل المحرر.
- **📊 سجل العمليات**: قناة خاصة (Output Channel) لمتابعة حالة العداد والتنبيهات.

---

## 🛠️ كيف تبدأ؟
1. بعد تثبيت الإضافة، ستظهر لك رسالة ترحيبية.
2. لتعديل الإعدادات:
   - افتح **Settings** (Ctrl + ,).
   - ابحث عن `Tasbeeh`.
   - قم بتغيير المدة الزمنية أو قائمة الأذكار كما تحب.
3. لتجربة التنبيه فوراً:
   - افتح الـ Command Palette (Ctrl + Shift + P).
   - اكتب `Tasbeeh: اعرض ذكر الآن`.

---

## 🗺️ خارطة الطريق (Roadmap) - الرؤية القادمة
نحن لا نبني مجرد "منبه"، بل نبني رفيقاً روحياً للمبرمج. إليك ما نخطط له في النسخ القادمة:

### 🟢 المرحلة القادمة (قريباً)
- **🔊 التنبيهات الصوتية**: خيار لتشغيل صوت خفيف وجميل مع التنبيه.
- **📂 فئات الأذكار**: إمكانية الاختيار بين (أذكار الصباح، أذكار المساء، أذكار الاستغفار).
- **🌍 تعدد اللغات**: دعم كامل للغة الإنجليزية والفرنسية بجانب العربية.

### 🟡 المرحلة المتقدمة
- **📈 إحصائيات الذكر**: لوحة تحكم بسيطة تظهر لك عدد الأذكار التي قرأتها خلال اليوم/الأسبوع.
- **🎯 أهداف يومية**: تحديد ورد يومي (مثلاً 100 استغفار) ومتابعة التقدم.
- **🌙 الوضع الليلي المطور**: تناسق ألوان التنبيهات بشكل أفضل مع كافة الـ Themes.

### 🔴 الرؤية البعيدة
- **📱 المزامنة**: ربط إحصائياتك مع تطبيق هاتف محمول لمتابعة وردك في كل مكان.
- **🤝 مجتمع الذاكرين**: خيار (اختياري) لرؤية عدد الذاكرين حول العالم في هذه اللحظة.
