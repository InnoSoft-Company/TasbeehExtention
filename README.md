# Tasbeeh Reminder

إضافة بسيطة لبرنامج VS Code تقوم بتذكيرك بذكر الله أثناء البرمجة. الإضافة تعرض رسالة منبثقة (Notification) تحتوي على ذكر أو تسبيح بعد مرور مدة زمنية محددة.

## المميزات
- إمكانية تحديد المدة الزمنية بين كل تنبيه (المدة الافتراضية 30 دقيقة).
- إمكانية تعديل الأذكار أو إضافة أذكار جديدة من خلال إعدادات VS Code.
- توفير أمر `Tasbeeh: اعرض ذكر الآن` لعرض تسبيح فوراً.

## كيفية الاستخدام
1. الإضافة تعمل تلقائياً بمجرد فتح VS Code.
2. لتغيير المدة الزمنية أو الأذكار:
   - اذهب إلى الإعدادات `Settings`.
   - ابحث عن `tasbeeh`.
   - قم بتعديل `Tasbeeh: Interval` لتغيير المدة (بالدقائق).
   - قم بتعديل `Tasbeeh: Messages` لتغيير قائمة الأذكار.

## كيفية تشغيل الإضافة على جهازك
1. افتح المجلد في VS Code.
2. اضغط `F5` لتشغيل الإضافة في نافذة تجريبية (Extension Development Host).
3. يمكنك تجربة عرض ذكر الآن بالضغط على `Ctrl+Shift+P` وكتابة `Tasbeeh: اعرض ذكر الآن`.

## لإنشاء ملف التثبيت (vsix)
إذا كنت تريد تثبيت الإضافة بشكل دائم أو مشاركتها:
1. قم بتثبيت الأداة الخاصة بالتغليف عن طريق موجه الأوامر:
   `npm install -g @vscode/vsce`
2. نفذ الأمر التالي داخل مجلد الإضافة:
   `vsce package`
3. سيتم إنشاء ملف بامتداد `.vsix` يمكنك تثبيته من خلال قائمة الإضافات في VS Code (Install from VSIX).
